O mnie

Cześć. Mam na imię Dawid. Uczę się programowania od kilku lat. Swoją wiedzę zdobywam z kursów, czytając dokumentację, zadając pytania na forach, szukając odpowiedzi na Stack Overflow, wykonując zadania rekrutacyjne itd. Chciałbym wreszcie spróbować swoich sił jako junior frontend developer. Indywidualna nauka nigdy nie będzie tym, co praca w zespole pod okiem doświadczonych programistów, dlatego czas to zmienić.

Umiejętności

Znam: HTML 5, JavaScript, TypeScript, ReactJS, Redux, REST API, CSS, SASS, BEM, Bootstrap 5, Webpack, Git, Linux

Do cięcia layoutów używam: Figmy lub Photoshop

Projekty

Najlepszym sposobem na ocenę moich umiejętności jest wgląd na kod. Każdy projekt posiada dwa linki. Jeden kieruje do podglądu na żywo, a drugi do zdalnego repozytorium na platformie Github, w którym znajduje się między innymi folder src, gdzie można szybko spojrzeć jak pracuję z danym językiem programowania.

Poniższe projekty zostały napisane w React:

  • React Animals Photo Search Engine - wyszukiwarka zdjęć zwierząt, korzystająca z Unsplash API
  • React People List - aplikacja do dodawania oraz usuwania elementów z listy
  • React Polluted Cities - aplikacja, która pobiera 10 najbardziej zanieczyszczonych miast w Polsce, Niemczech, Hiszpanii lub Francji. Wcześniej została napisana w języku javascript, teraz jest w React/TypeScript
  • React Form Builder - aplikacja do budowania formularzy. Wcześniej została napisana w języku javascript, teraz jest w React/TypeScript
  • React Currencies App - aplikacja, która pobiera najnowsze kursy walut. Do zarządzania stanem aplikacji użyłem Redux
  • React Week Note App - aplikacja do zapisywania, usuwania i edytowania notatek. Do zarządzania stanem aplikacji użyłem Redux
